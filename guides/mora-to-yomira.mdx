Mora and Yomira are separate products.

Mora maps the possible options. Yomira simulates reactions to concrete artifacts or options.

Use them together when the user is deciding what to write, publish, send, price, build, or launch.

## Workflow

1. Use Mora to map the option space.
2. Convert each promising path into a concrete artifact or decision option.
3. Use Yomira to simulate each option.
4. Compare distributions, raw voices, and decision implications.
5. Choose or revise based on evidence.

## Example

User asks:

```text
I want to message a potential user. I have three possible angles, but I do not know which one will land.
```

Agent should:

1. Use Mora to structure the message options.
2. Draft or ask for the exact message variants.
3. Run Yomira on every concrete variant.
4. Return the predicted reactions and recommended send strategy.
