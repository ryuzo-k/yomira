Agent Simulation API Create account

Simulate private human reactions, then export the result.

Use this before publishing content, launching an offer, sending a sales message, changing an X profile, or choosing between product directions.

Create account

Web UI

  1. Create an account and get trial credits.
  2. Open the dashboard.
  3. Paste the artifact, audience, and objective.
  4. Start with 40 simulated people for speed.
  5. Download Markdown or JSON for later work in Claude, Codex, Cursor, or your team.

API

curl -s -X POST "https://agent-simulation-api.vercel.app/api/simulate" \
  -H "content-type: application/json" \
  -H "x-api-key: $AGENT_SIMULATION_API_KEY" \
  -d '{
    "objective": "Decide whether to publish this content.",
    "artifact": {
      "type": "content_draft",
      "content": "Paste the actual text people will see."
    },
    "audience": {
      "description": "Describe who will see this."
    },
    "simulation": {
      "target_n": 40,
      "max_agent_voices": 8
    }
  }'

Download Results

curl -s "https://agent-simulation-api.vercel.app/api/simulations/SIMULATION_ID?format=markdown" \
  -H "x-api-key: $AGENT_SIMULATION_API_KEY"

Agent Skill

Use the repo skill at skills/agent-simulation/SKILL.md. Decision Foundation maps possible paths; Agent Simulation tests one path or artifact against likely human reaction.

Suggested Defaults

Fast: 40 people, 8 voices. Use this for first-pass content or messaging checks.

Serious: 120 people, 12 voices. Use this for launch decisions, pricing, and important offers.

Report: 180+ people. Use this when you plan to download and discuss the result later.
